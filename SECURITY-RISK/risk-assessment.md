Security risk assessment report 

Part 1: Select up to three hardening tools and methods to implement
Multi-factor Authentication (MFA)
Strong password policies and unique credentials
Firewall rules and network traffic filtering



Part 2: Explain your recommendations
The organization experienced a data breach due to several security vulnerabilities in their network and authentication systems. 
To improve security, I strongly suggest that the organization implements Multi-Factor Authentication (MFA) to mitigate the ease of brute force attacks.

Adding MFA will require an additional step beyond the password to gain access to the network. This reduces the risk of unauthorized access if a password is stolen or guessed during a brute force attack. 

There also needs to be a change to the password policies. Employees should no longer be allowed to share passwords. This is a major security risk because it increases the likelihood of unauthorized access . Additionally, default passwords should always be changed to strong, unique passwords to prevent attackers from easily gaining access to the systems. 

Finally, the organization should also implement firewall rules to filter incoming and outgoing network traffic. Firewalls help protect the network by allowing only authorized traffic and blocking suspicious connections.
